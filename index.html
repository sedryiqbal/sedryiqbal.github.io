KOTAK-STERIL-SINAR-UV-C
Berbasis Internet-Of-Things

Pada kotak Sterilisasi yang telah di buat sistem kinerja alat di mulai menggunakan aplikasi telegram untuk menghidupkan lampu uv-c dan penguncian pada kotak steril. Sistem sterilisasi berjalan selama 15 menit dengan kondisi lampu uv-c menyala dan kotak steril terkunci. Pada proses sterilisasi Sensor ML8511 akan terus membaca nilai lampu sinar uv-c dengan satuan J/cm2 untuk mengetahui sinar yang dihasilkan dari lampu uv-c tersebut.

Komunikasi Basestation Robot KRSBI BERODA

Base Station adalah komputer, notebook, atau laptop yang menjalankan software yang digunakan untuk memantau dan mengatur kerja robot secara otomatis berdasarkan instruksi yang diterima dari RefBox. Base Station melakukan koordinasi aktivitas setiap robotnya dalam pertandingan. Selama pertandingan sedang berjalan, anggota tim peserta tidak diizinkan untuk mengoperasikan Base Station-nya. Base Station menggunakan kabel Ethernet Switch untuk melakukan koneksi dengan RefBox. Base Station memanggil alamat IP (IP Address) RefBox untuk melakukan koneksi melalui Port 28097 (default).

Deteksi Bola dan Komunikasi Robot Python Multithreading

Deteksi Bola menggunakan metode hsv di python yang terhubung dengan robot sehingga dapat mengejar bola apabila terdeteksi bola orange di atas lapangan. Pada program ini menerapkan sistem multithreading memungkinkan lebih dari satu pengguna pada satu waktu tanpa memerlukan banyak salinan dari program yang berjalan di komputer. Multithreading juga dapat menangani beberapa permintaan dari pengguna yang sama, sehingga dalam sistem ini ada 2 kondisi berjalan yaitu deteksi bola dan komunikasi udp untuk menerima data dari aplikasi basestation berupa data char untuk perintah menggerakan robot secara otomatis.

MACHINE LEARNING PEMULA MEMBEDAKAN FOTO GUNTING-BATU-KERTAS

Machine Learning pemula membedakan foto gunting batu kertas, ada beberapa library yang digunakan dalam project ini dapat dilihat pada gambar 2. Ada 3 folder dalam dataset yang digunakan dan dataset juga di bagi menjadi data train ,data val dan data test. dataset yang sudah di bagi diproses terlebih dahulu sebelum di load menggunakan fungsi image data generator.Setelah selesai masuk ke model agar data dapat dilatih. Pada model kali ini menggunakan library keras sequential. Selesai data di latih dengan model maka epoch dapat di lihat tingkat accuracy nya, apabila sudah bagus nilai accuracy terakhir uji coba input foto gunting, batu dan kertas.

WEB-PELAYANAN-BEASISWA-DOSEN

Website Aplikasi Layanan Pemberian Rekomendasi BUDI (Beasiswa Unggulan Dosen Indonesia) di Lembaga Layanan Pendidikan Tinggi Wilayah II terdiri dari 11 halaman dan 3 buah tabel (tb_user, dokumendn, dokumenln) pada database beasiswa. Pengelolaan data pada aplikasi ini menghasilkan output berupa surat rekomendasi beasiswa dosen dengan melakukan pengajuan terlebih dahulu dan menginputkan berkas yang diminta sesuai dengan syarat & kententuan yang ada pada halaman user. Setelah melakukan input data pada form pendaftaran pengajuan beasiswa, admin akan memberi konfirmasi “ACC” dan dapat mencetak surat rekomendasi beasiswa tersebut, akan tetapi jika berkas belum memenuhi syarat & ketentuan maka admin akan memberi konfirmasi “Belum ACC” berupa note berkas yang harus di lengkapi .

Implementasi Sistem Aplikasi dan Database Penyimpanan Data Pada Perangkat Drywash Sanitizer Berbasis IOT

Drywash Sanitizer Bekerja dengan 2 Sensor Ultrasonik di dalam Kotak untuk mengetahui objek (tangan). Apabila sensor mendeteksi objek maka kotak akan menghasilkan embun dan proses sterilisasi berlangsung sampai selesai. Aplikasi berfungsi untuk menerima data water level yang berada di dalam kotak yang berisi air. Data di kirim masuk secara real time dan dalam bentuk grafik. Sebelum masuk ke aplikasi user harus register terlebih dahulu dan login.
